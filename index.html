Bem-vindo ao Projeto Demo JavaScript

Este projeto demonstra várias funcionalidades do JavaScript moderno em um único local. Explore as diferentes seções para aprender sobre manipulação do DOM, eventos, arrays, timers, armazenamento local e muito mais.

        Documentação Curta

        O projeto demonstra várias APIs do navegador sem dependências externas.

        Funcionalidades Principais

        • Manipulação do DOM: Adição, remoção e edição de elementos
        • Eventos: Clique, arrastar, soltar, entrada de formulário
        • Arrays: Operações com map, filter e reduce
        • Timers: setTimeout, setInterval e Math.random
        • Armazenamento: localStorage para persistência de dados
        • Animações: Transições CSS e efeitos visuais
        • CRUD: Operações Create, Read, Update, Delete
        • Drag & Drop: API nativa de arrastar e soltar
        • Galeria: Navegação por imagens com miniaturas
        • To-Do: Lista de tarefas com persistência
        • Progresso: Barra de progresso animada
        • Estatísticas: Dados em tempo real
        • Depuração: Painel de logs integrado

        Como Usar

        1. Abra o arquivo index.html em um navegador moderno
        2. Navegue pelas diferentes seções usando o menu
        3. Interaja com os controles em cada seção
        4. Observe os resultados e logs em tempo real
        5. Experimente alternar entre os temas claro e escuro
